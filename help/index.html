Kerberized Certificate Authority Provider for Network Identity Manager

The Kerberized Certificate Authority Provider for Network Identity Manager provides the ability for Network Identity Manager to obtain and manage KCA certificates.

The installation package includes the KCA Provider for Network Identity Manager (kcacred.dll) as well as an PKCS #11 module which can be used with the Mozilla family of products (kpkcs11.dll) that exposes the Windows "My Certificates" store. Instructions for installing the PKCS #11 module can be found here.

During normal operation, the KCA provider for Network Identity Manager will obtain new KCA certificates (as configured during the new credentials acquisition process) and place them in the Windows certificate store. Once placed there, applications that use certificates from this store will be able to use the KCA certificates. However, the Mozilla family of products requires the installation of the PKCS #11 module to be able to use these certificates.
